# Analyzing Public Transit Impact on Chicago Property Values

Welcome to the GitHub repository for the project **Analyzing Public Transit Impact on Chicago Property Values**.

This project quantitatively assesses the influence of the Chicago Transit Authority (CTA) rail system (L-stops) on residential property values within Cook County, aiming to provide data-driven insights for urban planning and real estate investment.
